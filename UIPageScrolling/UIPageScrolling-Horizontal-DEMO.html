UI Page Scrolling
Плагин позволяет создавать веб-странницы в виде слайдов
Скроллинг можно осуществлять как сверху вниз, так и слева направо
Пример вертикального скроллинга Перейти на GitHub К списку компонентов
Начинаем работу
Для работы с плагином, необходимо установить HTML разметку:
<div class=".main">
    <section>
        /some code/
    </section>
    <section>
        /some code/
    </section>
</div>
Затем, с помощью вызова фунции
$(".main").UIPageScrolling();
Будет получена веб-страница с 2 слайдами. Контент содержится в section

При необходимости, можно создать элементы, которые будут использоваться в роли переключателей слайдов. (Как на этой странице - справа) Для этого необходимо создать необходимое количество элементов (например, задав им класс page-control) и вызвать функцию, задав селектор на page-control:
$(".main").UIPageScrolling({sectionsControl:".page-control"});
Для отображения активной страницы, задайте стили классу: ui-page-scrolling-control_active
Поддерживаемые настройки
Функция UIPageScrolling в качестве параметров принимает JSON объект
Возможные параметры:
{String} sections : пользовательский селектор (по умолчанию section)
{String} easing : transition-timing-function
{Number} time : время анимации
{Function} beforeMoveFunc : функция, исполняющаяся перед анимацией
{Function} afterMoveFunc : функция, исполняющаяся после анимации
{Boolean} isCyclic : цикличная прокрутка
{Boolean} isVertical : вертикальная\горизонтальная прокрутка
{String} sectionsControl : селектор на выбор слайда
{Boolean} captureKeyboard : захват клавиатуры (управления курсором + Page Up\Down)
{Boolean} captureTouch : захват скроллинга пальцем (мышью) по умолчанию - отключено (false)
