    Описание

    Компонент, модифицирующий UICarriage и предлагающий быстрое создание скроллинга для контента.

    Поддерживается как вертикальный, так и горизонтальный скроллинг.

    Главная особенность - возможность кастомизировать скролл любым удобным образом.

    Быстрый старт

    UIScroll.create([HTMLElement || jQuery], params)

    Например:

    UIScroll.create($('.some-carriage-selector'), {$target: $('.some-content-selector')})

    Параметры и методы

    UIScroll является расширением компонента UICarriage и предоставляет "наружу" один метод:

    • resize - оповещает компонент о том, что размеры области для скролла могли измениться и поэтому необходимо пересчитать показатели

    Параметры для uiCarriage.scroll:

    • {Boolean} hideOnLeave - скрывать ли скрол, если мышь уходит за область видимости контента (по умолчанию true)
    • {String} scrollEndEvent - Событие генерируемое при окончании скролла (по умолчанию 'scroll:stop')
    • {String} scrollEvent - Событие генерируемое при скролле (по умолчанию 'scroll:move')
    • {Boolean} isHorizontal - горизонтальный ли скролл (по умолчанию false)
    • {JQuery} $target - jQuery объект элемента, контент которого должен скролиться. Должен содержать в себе один элемент который и будет скроллиться
    • {Number} minSize - минимальный размер скроллинга (по умолчанию 30)
    • {Boolean} preventScroll - событие скролла не будет скроллить контент (запрещает стандартное поведение)

    UIScroll, по умолчанию, при окончании сколла триггерит событие 'scroll:stop' и при движении 'scroll:move' на элементе-инициализаторе
